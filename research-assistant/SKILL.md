# Skill: Research Assistant
## Fonctions
- **`search_papers <query>`** : Cherche des papers sur arXiv.
- **`summarize_paper <url>`** : Résume un paper depuis une URL ou un PDF.
- **`generate_report <topic>`** : Génère un rapport Markdown sur un sujet.
- **`analyze_data <file>`** : Analyse un fichier de données (CSV, JSON).
- **`cite_paper <doi>`** : Génère une citation au format APA/MLA.
